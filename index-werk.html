projecten

Ik heb aan een aantal projecten gewerkt waar ik zeer op trots ben, de projecten waaraan ik heb deelgenomenzijn: Onderzoek Schermen Pathe, Kook app over burgers, albert heijn WK bier kratten website design. Voor het pathe project kreeg ik samen met mijn team de opdracht van pathe om te onderzoeken waarom de ticket machines niet veel gebruikt worden en hoe dit verbeterd kan worden. bij dit project hebben we verschillende methodes gebruikt om het onderzoek te verrichten , er is desk research gedaan, ook ben ik naar de pathe zelf gegaan om klanten te interviewen over de ticket machine. Een ander project waaraan ik heb deelgenomen was het de opdracht om een kook app te maken over burgers. bij dit project moesten we de burgers zelf ook maken om van uit de users perspectief te kijken. voor de burger app hebben we de apps figma en marvel gebruikt. bij het project van Albert Heijn kreeg ik de opdracht om een website te designen en een prototype hiervan te maken over wk bier kratten waar de klant trie verschillende bier kratten kan bestellen. bij dit project heb ik alleen figma gebruikt om de website te maken.
